SVG favicons for my projects, served via GitHub Pages.

## Usage

Reference a favicon in your HTML:

```html
<link rel="icon" type="image/svg+xml" href="https://<username>.github.io/favicon-farm/<project>.svg">
```

## Adding a favicon

Drop an SVG file named `<project>.svg` in the root of this repo.
