# favicon-farm

A central place for all my project favicons — SVGs served via [GitHub Pages](https://mayankvachher.github.io/favicon-farm/).

| www | [www.svg](https://mayankvachher.github.io/favicon-farm/www.svg) | `https://mayankvachher.github.io/favicon-farm/www.svg` |

## Usage

Add this to your HTML `<head>`:

```html
<link rel="icon" type="image/svg+xml" href="https://mayankvachher.github.io/favicon-farm/<project>.svg">
```

## Adding a new favicon

1. Drop an SVG file named `<project>.svg` in the root of this repo.
2. Update the table above.
3. Push to `main` — GitHub Pages will deploy automatically.
