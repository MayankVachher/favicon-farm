## Usage

Add this to your HTML `<head>`:

```html
<link rel="icon" type="image/svg+xml" href="https://favicon-farm.mynkv.ca/<project>.svg">
```

## Adding a new favicon

1. Drop an SVG file named `<project>.svg` in the root of this repo.
2. Update the table above.
3. Push to `main` — GitHub Pages will deploy automatically.
